#### US-1.1 : Message de bienvenue centré

**Statut** : ✅ done (2026-02-09)

**En tant que** visiteur du site

**Je souhaite** voir un message de bienvenue sur la page d'accueil

**Afin de** savoir que j'ai atteint la galerie des souvenirs et être accueilli de façon claire.

- **Critères d'acceptation** :
- **CA1 - Titre** :
  - Le message affiché est « Bienvenue sur la galerie des souvenirs » (visiteur non connecté) ou « [prénom], bienvenue sur la galerie des souvenirs » (visiteur connecté, si prénom disponible).
  - Le texte est rendu en tant que titre H1 (balise sémantique h1).
- **CA2 - Centrage** :
  - Le titre est centré horizontalement dans la zone de contenu visible.
  - Le titre est centré verticalement dans la zone de contenu visible (viewport ou zone principale).
  - Le centrage est conservé sur les tailles d'écran usuelles (mobile, tablette, desktop).
